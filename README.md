# 👑 Charles le Bogosse

Le boss absolu qui gère tout avec style 😎

## Description

Une page web simple en React qui célèbre Charles, le bogosse légendaire.

### Features

- ⚡ React + Vite pour un dev ultra-rapide
- 🎨 Design moderne avec gradient et animations
- 📱 Responsive design
- 🚀 Serveur Express pour la prod
- 🔥 Compteur de niveau de bogosse

## Installation

```bash
npm install
```

## Dev

```bash
# Lance Vite en mode dev (hot reload)
npx vite

# Serveur Express (dev)
npm run dev
```

L'appli sera accessible sur http://localhost:5173 (Vite) ou http://localhost:3000 (Express en dev).

## Build & Production

```bash
# Build pour la production
npm run build

# Lance le serveur en production
npm start
```

## Tech Stack

- **Frontend**: React 18
- **Build tool**: Vite
- **Server**: Express
- **Styling**: CSS pur (pas de framework)
